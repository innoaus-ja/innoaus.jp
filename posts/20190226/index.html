C++コンソールアプリケーションに色をつける
TIP

プロジェクトrangはヘッダファイルのみで動作するコンソールアプリケーションに色をつけるオープンソースです。
https://github.com/agauniyal/rang

Colors for your Terminal.

Unix-like (Linux, unix, mac)

Windows Demo

必要ライブラリ

rangはC++ standard libraryのみで動作するため、追加ライブラリは必要ありません。

インストール

$ git clone https://github.com/agauniyal/rang.git

適用

インストールフォルダのinclude/rang.hppファイルをコピーするか、includeフォルダをプロジェクトパスに追加します。

簡単なsample code

#include "rang.hpp"
int main()
{
    rang::setControlMode(rang::control::Auto); 
    // rang::control::Auto　//コンソルが色をサポートするかを自動検出
    // rang::control::Off // 適用解除
    // rang::control::Force //強制適用
    
    std::cout << "Hello world !!!" 
              << rang::style::bold 
              << "Rang styled text !!!"
              << rang::style::reset << std::endl;
    return 0;
}
