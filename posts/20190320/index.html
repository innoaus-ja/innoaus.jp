Onion Omega2+ Oled ExpansionでQRコードを表示する
Omega IoT Embedded

Omega2+ セットアップ方法はこちら

準備

  1. Onion Omega2+
  2. Expansion Dock
  3. OLED Expansion

ステップ

Omega2+ 接続

  1. ブラウザにomega-xxxx.local(xxxはOmegaMACアドレス下4桁)を入力し、Omegaにログインします。
  2. Terminalを選択します。
  3. login, passwordはログイン時使用したUsername, Passwordと同じです。

Open Packageインストール

opkgで必要なパッケージをインストールすることができます。

まず、opkgをアップデートします。

opkg update

opkgから必要なパッケージをインストールします。

opkg install python-light python-codecs pyOledExp git git-http ca-bundle

oled-qr-code-generatorダウンロード

gitコマンドでOnionから提供されているoled-qr-code-generatorをダウンロードします。 https://github.com/OnionIoT/oledQrCodeGenerator

cd
git clone https://github.com/OnionIoT/oledQrCodeGenerator.git

QRコード表示

QRコード内容をmain.pyにパラメータとして渡すとOLEDにQRコードが表示されます。

cd oledQrCodeGenerator
python main.py 'Hello, Onion?'
