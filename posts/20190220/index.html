vuepressを試す（基本設定:メニュー、サイドバー、meta要素設定方法）
vuepress

vuepressを試す（環境準備）の続き

サイト共通設定　ナビバー、サイドバー

dev_docsフォルダにabout.mdを作成します。
.vuepressフォルダにpublicフォルダとconfig.jsファイルを作成し、以下を記入。
publicフォルダには静的ファイル（画像など）を配置します。

module.exports = {
    themeConfig: {
        logo : 'logo.png', // \dev_docs\.vuepress\public\logo.png
        search: true, // 検索機能
        nav: [ //ナビバーリンク
            { text: 'Home', link: '/' },
            { text: 'About', link: '/about.html' },
            { text: 'サブメニュー',
                items: [
                    { text: 'メニュー１', link: '/' },
                    { text: 'メニュー２', link: '/about.html' },
                ]
            }
        ],
        sidebar: [　//サイドバーリンク
            ['/', 'サイドバーメニュー１'],
            ['/about.html', 'サイドバーメニュー２'],
        ]
    }
}

npm run dev

ブラウザhttp://localhost:8080/で確認します。
config.jsを修正した内容はリアルタイムで反映されないので,都度npm run devを実行する必要があります。

ページ別meta要素設定

ページ別のtitle, descriptionなどのmeta要素はmdファイル上部にyaml形式で書いておくと反映されます。（---で囲む）
＊config.jsのlocalesからも設定可能。

---
title: ページタイトル
description: ページの説明文
meta:
- name: keywords
  content: キーワード
- property: og:image
  content: http://localhost:8080/logo.png
- property: og:desciption
  content: ページの説明文
---

以下のように変換されます。

<title>ページタイトル</title>
<meta name="description" content="ページの説明文">
<meta name="keywords" content="キーワード">
<meta property="og:image" content="http://localhost:8080/logo.png">
<meta property="og:desciption" content="ページの説明文">
